# [convtr.io](http://convtr.io/) community

# Community Guidelines

Welcome to the [convtr.io](http://convtr.io/) community! This is a place where you can report issues, and request new features.

## Reporting Issues

If you encounter a problem while using [convtr.io](http://convtr.io/), we encourage you to report it. Please create a new issue in the issues tab, providing as much detail as possible about the problem you're experiencing. This will help us to address the issue more effectively.

## Requesting Features

We are always looking to improve [convtr.io](http://convtr.io/) and we welcome your ideas for new features. If you have a suggestion, please create a new issue in the issues tab, using the "Feature Request" template.

## Contributing

While [convtr.io](http://convtr.io/) is a closed source project, we welcome contributions from the community. If you have a fix for an issue or a new feature you'd like to add, you can raise a pull request (PR) against this repo.

Please note that your PR should include a detailed description of the changes you've made and why you believe they would be beneficial to [convtr.io](http://convtr.io/).

Thank you for being a part of our community!

## Conduct

As part of our community, we expect all members to treat each other with respect and courtesy. Please refrain from using offensive language, making personal attacks, or engaging in any form of harassment. Any behavior that disrupts the positive environment of our community will not be tolerated and may result in your removal from the community.
